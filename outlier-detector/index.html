In scientific research, an outlier is a data point that differs significantly from other observations. Identifying outliers is critical in animal studies and clinical trials to ensure statistical validity.

Methods Used

  • IQR Method: Uses the Interquartile Range to detect outliers. Robust for non-normal distributions. Any point falling below Q1 - 1.5*IQR or above Q3 + 1.5*IQR is flagged.
  • Grubb's Test: A statistical test used to detect a single outlier in a univariate data set that follows an approximately normal distribution.

Note: Always report whether outliers were removed and the method used in your "Methods" section.
