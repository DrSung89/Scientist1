üìä Outlier Detection: Auto-Selection Algorithm

This tool automatically runs a Shapiro-Wilk Normality Test first, then selects the appropriate statistical method based on the P-value.

1. If Data is Normal (P ‚â• 0.05) ‚Üí Grubb's Test

Grubb's test is the gold standard for detecting exactly one outlier in a normal distribution.

  • How it works: It calculates a G score (distance from the mean relative to SD).
  • Stepwise Removal: Grubb's test detects only one outlier at a time. If an outlier is found, remove it and re-run the test to check for a second one.

2. If Data is Non-Normal (P < 0.05) ‚Üí IQR Method

When data is skewed or the sample size is too small to prove normality, the Interquartile Range (IQR) method is more robust.

Thresholds:
Lower < Q1 - 1.5√óIQR   |   Upper > Q3 + 1.5√óIQR
  • Multiple Outliers: IQR can detect multiple outliers at once.
  • Warning: If multiple outliers are flagged, remove the most extreme value first and re-check, as removing data affects the quartiles.

üêÅ Human Equivalent Dose (HED)

This calculator converts doses between species based on Body Surface Area (BSA) normalization.

Planning Animal Studies from Human Doses:
When calculating a starting dose for animal experiments based on a human approved dose (e.g., a flat dose in mg), you must first convert it to mg/kg.

‚ö†Ô∏è Important Note:
For this conversion, use the FDA standard human body weight of 60 kg. Do not use arbitrary weights unless specific data is available.

Step 1 (Human Dose to mg/kg):
Human Dose (mg/kg) = Total Human Dose (mg) / 60 kg

Step 2 (Conversion to Animal Dose):
Animal Dose (mg/kg) = Human Dose (mg/kg) √ó (Human Km / Animal Km)
