üìâ T-Test Calculator (Robust)

Performs an Independent Samples T-Test. By default, our calculator uses Welch's T-Test.

Why Welch's T-Test?

Unlike standard Student's t-tests, Welch's t-test does not assume equal variances between groups. This makes it the safer, more robust choice for real-world biological data.

  • P-value: Indicates statistical significance (typically < 0.05).
  • Cohen's d: Measures Effect Size. 0.2 (Small), 0.5 (Medium), 0.8 (Large).

‚ö†Ô∏è Low Count Validation

‚è≥ Survival Analysis (Kaplan-Meier & Log-Rank)

Generates survival curves and calculates statistics for time-to-event data (e.g., Overall Survival in oncology).

  • Event (Code 1): The event occurred (e.g., death, progression). The curve will drop.
  • Censored (Code 0): The subject was lost to follow-up or the study ended. The curve does not drop.
  • Median Survival: The time point where survival probability reaches exactly 50%. If the curve never drops that low, it is marked as "Not Reached".

üìä Log-Rank Test

For 2 or more groups, a Log-Rank Test is automatically performed to generate a P-value, determining if the survival curves are significantly different.
